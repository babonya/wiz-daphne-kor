# Antigravity Daphne 프로젝트 수행 규칙

- 코딩이나 컴파일 등 소스코드 또는 프로젝트 환경에 실질적인 변화를 가하는 작업을 수행하기 전에는, 항상 구현 방향과 작업 내용을 사용자에게 설명하고 먼저 논의해야 합니다.
- 논의가 끝난 후, 실제로 코딩 작업을 진행해도 될지 동의를 구하는 확인 단계를 거쳐 사용자의 확인을 얻은 뒤에 코딩을 수행합니다.
- 매크로 프로그램 버전을 업그레이드(버전업)할 때마다, 메인 폴더의 `Daphne Antigravity.bat` 및 릴리즈용 `run.bat` 파일 내 `title` 버전을 항상 최신 버전에 맞춰 함께 업데이트해야 합니다.
